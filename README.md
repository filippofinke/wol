## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/filippofinke/wol.git
   ```
2. Install NPM packages
   ```sh
   npm i
   ```
3. Configure the .env file

## Usage

1. Start the server
   ```sh
   npm start
   ```
